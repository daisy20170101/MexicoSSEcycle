Files gps_[year].txt contain data from geodetic observations, whereas files synthetic_GPS[year].txt contain data from our modeled displacements. The columns in the files show:
# COLUMN  DATA                    UNITS
1        GPS station longitude     degrees
2        GPS station latitude      degrees
3        East displacement         mm (m)*
4        North displacement        mm (m)
5        Vertical displacement     mm (m)

*Units in parenthesis correspond to those of the synthetic data.
